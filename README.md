### 启动源码

**基本**

- Node.js v16

---

**获取代码**

```console
> git clone https://github.com/nekoMC/NekoSite.git
> cd NekoSite
> npm install
```

---

**启动**

```console
> npm run start
```

---

**构建**

构建当前平台的发布包

```console
> npm run build
```
